# 💻 Laptop Price Prediction Using KNN Regressor

## 📌 Project Overview

This project predicts laptop prices based on specifications such as company, product type, screen size, RAM, storage, processor, and weight using a K-Nearest Neighbors (KNN) Regressor model.

The application is deployed using Streamlit and provides real-time laptop price predictions through an interactive web interface.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

---

## 📊 Machine Learning Workflow

1. Data Cleaning
2. Feature Engineering
3. Data Preprocessing
4. Feature Transformation
5. KNN Regressor Model Training
6. Model Evaluation
7. Streamlit Deployment

---

## 📁 Project Files

- `app.py` → Streamlit Application
- `model.pkl` → Trained KNN Model
- `transformer.pkl` → Data Preprocessing Transformer
- `requirements.txt` → Required Libraries
- `laptop_price.csv` → Dataset
- `KhajaMainuddin_Laptop_Price_Prediction.ipynb` → Complete Notebook
